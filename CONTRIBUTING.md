# Contributing to ReconForge

Thank you for your interest in contributing to ReconForge.

ReconForge is designed as a professional, intelligence-driven CLI reconnaissance framework focused on structured output, deterministic behavior, and offensive security workflows.

We welcome contributions that improve reliability, modularity, and intelligence capabilities.

---

## Project Philosophy

ReconForge prioritizes:

- Clean CLI-first design
- Structured intelligence extraction
- Deterministic and reproducible output
- Minimal noise
- Modular and extensible architecture
- Professional-grade reporting

Please keep these principles in mind when contributing.
